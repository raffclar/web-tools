# Web Tools Collection

A bunch of tools I've built to help with developing. [Visit the live site.](https://blog.zanidrak.com/web-tools/)

# Developer Tools
- [JWT Debugger & Generator](https://blog.zanidrak.com/web-tools/jwt) - Decode, verify and generate JSON Web Tokens
- [Amplify Tools](https://blog.zanidrak.com/web-tools//amplify/) - Similar to the JWT tool but with additional parsing for Cognito tokens
- [UUID Generator](https://blog.zanidrak.com/web-tools//uuid/) - Generate random UUIDs/GUIDs
- [Date & Time Utilities](https://blog.zanidrak.com/web-tools//date/) - Get an ISO8601 or Unix epoch timestamp value
- [Benchmark Tool](https://blog.zanidrak.com/web-tools//benchmark/) - Performance testing utilities
- [Glitcher](https://blog.zanidrak.com/web-tools//glitcher/) - Create glitch effects for images
- [Diffusion](https://blog.zanidrak.com/web-tools//diffusion/) - View embedded metadata of AUTO111 & co. generated images
- [LoRa Tools](https://blog.zanidrak.com/web-tools//lora/) - Image generation LoRa-related utilities. View the metadata within LoRA files
- [Magic Eight-Ball](https://blog.zanidrak.com/web-tools//eight-ball/) - Ask a question and get a classic Magic Eight-Ball answer
